### 수정 항목 상세

#### 1-A. 제목 — `우주가 다름` (오류)

| | 내용 |
|--|------|
| 현재 | `「필링」이에요 「필러」예요? — 한 글자 차이, 시술은 우주가 다름` |
| 제안 | `「필링」이에요 「필러」예요? — 한 글자 차이, 하는 일은 완전히 다릅니다` |
| 이유 | 「우주가 다름」은 한국어 관용 표현에 없음. 생성 AI 오류 또는 동음 혼동으로 추정. |

#### 1-B. 본문 15–17행 — `탭 A / 탭 B` (UI 은유)

| | 내용 |
|--|------|
| 현재 | `두 개의 탭이 동시에 뜹니다. / 탭 A: 필링 — … / 탭 B: 필러 — …` |
| 제안 | `두 가지 방향으로 나뉩니다. / 필링 — … / 필러 — …` |
| 이유 | 브라우저 UI 비유(탭)가 진료 안내 문체와 이질적. 정보 전달에 불필요. |

#### 1-C. 본문 19행 — `가르면` (오타)

| | 내용 |
|--|------|
| 현재 | `받침과 느낌 — 가르면, 하는 일이 완전히 다릅니다` |
| 제안 | `받침과 느낌 — 구분하면, 하는 일이 완전히 다릅니다` |
| 이유 | `가르면`은 「쪼개다」 뜻. 문맥상 「구분하면」이 정확함. |

#### 1-D. 본문 53행 — `상담의 플롯` (서사 용어)

| | 내용 |
|--|------|
| 현재 | `그 한 마디에 상담의 플롯이 반은 정해집니다` |
| 제안 | `그 한 마디에 상담 방향의 절반은 결정됩니다` |
| 이유 | `플롯`은 드라마·소설 용어. 의료 상담 맥락과 레지스터 불일치. |
