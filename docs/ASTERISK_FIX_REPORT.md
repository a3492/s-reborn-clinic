## 제거된 마크다운 패턴 유형

### 1. `**굵은 텍스트**` (이중 별표)

가장 많이 등장한 패턴. AI 생성 글에서 핵심 단어·구 강조에 과도하게 사용됨.

예시 (제거 전 → 후):

| 파일 | 수정 전 | 수정 후 |
|------|---------|---------|
| `procedures/botox-natural-guide.md` | `**보톡스**는 보툴리눔` | `보톡스는 보툴리눔` |
| `myth/peeling-not-filler-myth.md` | `**철자 한 글자** — 정확히는 받침` | `철자 한 글자 — 정확히는 받침` |
| `faq/first-consultation-faq.md` | `**마취 크림**을 도포합니다` | `마취 크림을 도포합니다` |
| `health-tips/skincare-ingredients-guide.md` | `**레티놀**은 비타민 A` | `레티놀은 비타민 A` |

### 2. `*기울임 텍스트*` (단일 별표)

주로 출처 표기·특수 어조 구간에 사용됨. 제거 후 일반 문장으로 통일.

예시 (제거 전 → 후):

| 파일 | 수정 전 | 수정 후 |
|------|---------|---------|
| `clinic-news/thermage-key-doctor.md` | `*출처: 팜뉴스 2024.03.06*` | `출처: 팜뉴스 2024.03.06` |
| `myth/botox-stop-responding-myth.md` | `*기간이 짧다*`, `*거의 효과가 없다*` | `기간이 짧다`, `거의 효과가 없다` |

## 보존 확인 항목

| 보호 항목 | 확인 결과 |
|---------|----------|
| frontmatter `title`, `description`, `tags` 내 따옴표 | 변경 없음 |
| 코드 블록 내 `*` 패턴 | 변경 없음 |
| 마크다운 링크 `[텍스트](url)` | 변경 없음 |
| 불릿 리스트 `- 항목` 앞 `- `(단독 별표 없음) | 해당 없음 |
| 수평선 `---` | 변경 없음 |
