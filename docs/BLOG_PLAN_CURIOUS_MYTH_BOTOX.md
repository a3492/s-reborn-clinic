# 기획서: MYTH 글 1편 — 독자 흥미 유지형

## 1. 목적

- **글 유형**: `myth` (MYTH · 자주하는 오해)
- **기능**: 오해를 풀되, 끝까지 읽게 만들고 **다음 글·재방문**으로 이어지게 설계
- **기존 글과 차별**: `procedures/botox-natural-guide.md`는 시술 안내, 본 글은 **“왜 그런 말이 돌까?”**에 초점

## 2. 타깃 독자

- 보톡스를 고려 중이나 “표정 굳는다”는 말에 망설이는 20~50대
- SNS·지인 말에만 의존해 판단하려는 사람

## 3. 핵심 메시지 (한 줄)

> “보톡스 = 표정이 사라진다”는 **과장된 단편**이고, 자연스러움은 **용량·부위·숙련**의 문제다.

## 4. 흥미를 이어가는 기법 (본문에 반영)

| 기법 | 적용 |
|------|------|
| **도입 질문** | 첫 문단에서 독자의 속마음을 직접 인용형으로 질문 |
| **미니 서스펜스** | “그런데 여기서 한 가지 더…”로 단락 전환 |
| **부분 해결 → 남은 질문** | 오해의 절반은 풀고, 나머지는 의도적으로 다음 주제로 연결 |
| **시리즈 약속** | 말미에 “다음 MYTH에서는 ○○을 짚겠다” 명시 (구체 키워드) |
| **행동 CTA** | 막연한 홍보 대신 “체크리스트 3가지”로 스스로 점검 유도 |

## 5. 예상 목차

1. 도입: 들어본 적 있는 말들
2. 오해의 핵: “다 굳는다”는 말이 생긴 이유
3. 의학적으로 어떤 때 자연스러운가
4. 스스로 점검할 질문 3가지 (상담 전)
5. 마무리 + 다음 편 예고: “내성·간격·브랜드” 중 하나를 택해 후속 편 제목 후보로 연결

## 6. 톤 & 주의

- 진단·처방 대신 **일반 정보 + 내원 권유** 문구 유지
- 과장 치료 효과 표현 지양
